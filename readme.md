# Sitecore.Support.173445
This patch replaces `<p></p> ` with `<p>&nbsp;</p>` after saving the `raw value` of the `Rich Text Editor` field in `Content Editor` if the `HtmlEditor.RemoveScripts` setting is set to `true`.
